Previews and code

Previews and code

Previews are rendered in sandboxed iframes and are resizable.

Custom resizing currently only works properly in Firefox. In WebKit browsers you can only enlarge the previews..

There is currently no syntax highlighting of neither CSS nor HTML code, but it's on the todo-list.

Typography and text styles

Typography and text styles

We use the generic sans-serif font family to get Arial on Windows and Helvetica on Mac. The body font-size is set to 14px.

Paragraphs, lists, tables and blockquotes get some bottom margins as well as side margins.

Headings get font-sizes at a modular scale of 5:6.

  font-family: sans-serif;

  line-height: 1.618;

Links

Links

To show pseudo selectors styles in previews, add the pseudo selector (including the :) as a regular class. 

a {
  color: hsl(203, 90%, 50%);
  text-decoration: none;
}
  a:hover {
    color: hsl(203, 95%, 40%);
    text-decoration: underline;
  }
  a:visited {
    color: hsl(203, 100%, 35%);
  }
